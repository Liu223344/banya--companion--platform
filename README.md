# 伴芽儿童成长陪伴平台 MVP

伴芽是一个连接家长与认证陪伴者的儿童成长陪伴平台。本版本已经从静态展示页升级为可运行的网站 MVP，包含前端页面、Node.js 后端接口和本地 JSON 数据库。

## 本地运行

```bash
npm start
```

启动后访问：

```text
http://localhost:3000
```

## 当前功能

- 注册 / 登录
- 家长发布陪伴需求
- 家长选择推荐陪伴者并下单
- 陪伴者完善主页
- 陪伴者查看需求并接单
- 订单状态流转：已接单、陪伴中、已完成
- 订单消息沟通
- JSON 文件持久化数据
