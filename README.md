# 伴芽儿童成长陪伴平台 MVP

伴芽是一个连接家长与认证陪伴者的儿童成长陪伴平台。本版本已经从静态展示页升级为可运行的网站 MVP，包含前端页面、Node.js 后端接口和本地 JSON 数据库。

## 本地运行

```bash
npm start
```

启动后访问：

```text
http://localhost:3000
```

## 当前功能

- 注册 / 登录
- 家长发布陪伴需求
- 家长维护孩子档案
- 家长选择推荐陪伴者并下单
- 陪伴者完善主页
- 陪伴者提交认证申请
- 陪伴者查看需求并接单
- 订单状态流转：已接单、陪伴中、已完成
- 陪伴者提交陪伴记录
- 家长完成订单后评价陪伴者
- 订单消息沟通
- JSON 文件持久化数据

## 页面体验特性

- **暗色主题**：自动跟随系统设置，也可手动切换（顶栏太阳/月亮图标）
- **大字号模式**：顶栏 A 按钮一键放大全文，方便长辈和视障用户
- **响应式布局**：手机 / 平板 / 桌面三套断点，侧栏抽屉式折叠
- **微动效**：按钮 hover 微位移、卡片悬浮光效、视图切换淡入淡出
- **加载反馈**：按钮 loading 状态、骨架屏、全屏加载层
- **快捷操作**：`Ctrl/⌘ + K` 打开命令面板，快速跳转页面、切换主题和发布需求
- **可访问性**：键盘焦点环、跳转链接、`prefers-reduced-motion` 支持、tooltip 提示
